There are 4 kinds of selectors in CSS

  1. CSS Element Selector
  2. CSS ID Selectors
  3. CSS Class Selectors
  4. CSS Grouping Selectors
  • p tag is element selector
  • # pound is ID Selector
  • . dot is Class Selector
  • p, span, a is grouping selector
