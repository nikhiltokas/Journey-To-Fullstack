How CSS works?

Syntax is stored in CSS

For ex: P tag in HTML will be used as selector in CSS

There can be multiple selectors in class

We also use PROPERTY and VALUE in CSS


For ex: Color = Property, Blue = Value

Example 1: p {color: blue;}


There can be multiple properties as well

For ex: header, a, p.intro {background-color: blue; border-radius: 3px}


There are 3 kinds of CSS to the Markup/HTML

Inine CSS <p style="_____"> used directly in elements

Internal CSS <head style="_____"> used in head tags

External CSS keeping a seperate .css file altogether
